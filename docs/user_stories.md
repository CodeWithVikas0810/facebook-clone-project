# User Stories

## Authentication

### User Registration

As a user,
I want to register on the platform,
so that I can create an account and access the application.

### User Login

As a user,
I want to log in using my credentials,
so that I can access my profile and features.

### User Logout

As a user,
I want to log out of my account,
so that my account remains secure.

---

## Profile Management

### View Profile

As a user,
I want to view my profile,
so that I can see my personal information and activity.

### Update Profile

As a user,
I want to update my profile details,
so that my information stays current.

---

## Posts

### Create Post

As a user,
I want to create a post,
so that I can share updates with others.

### Edit Post

As a user,
I want to edit my post,
so that I can correct mistakes or update information.

### Delete Post

As a user,
I want to delete my post,
so that I can remove unwanted content.

### View Feed

As a user,
I want to view posts from users I follow,
so that I can stay updated.

---

## Likes

### Like Post

As a user,
I want to like a post,
so that I can show appreciation.

### Unlike Post

As a user,
I want to remove my like from a post,
so that I can change my reaction.

---

## Following System

### Follow User

As a user,
I want to follow another user,
so that I can see their posts in my feed.

### Unfollow User

As a user,
I want to unfollow another user,
so that I can stop receiving their updates.

---

## Comments

### Add Comment

As a user,
I want to comment on a post,
so that I can participate in discussions.

### Reply to Comment

As a user,
I want to reply to an existing comment,
so that I can have threaded conversations.

### Delete Comment

As a user,
I want to delete my comment,
so that I can remove unwanted content.

---

## Authorization

### Manage Own Content

As a user,
I want to edit or delete only my own posts and comments,
so that other users cannot modify my content.

### Admin Access

As an administrator,
I want to manage users and content,
so that I can maintain the platform.
